Detailed Description

Open addressing, quadratically probed hash tables.

All keys are kept in a single array with 2^n entries. A key is first looked up at index hash(k) mod 2^k, then following a quadratic sequence of offsets. The quadratic probing has decent locality of reference for the first few probes.

The search terminates when the key is found or a NULL is encountered. A special ERASED marker is used to avoid severing probe sequences when erasing keys.

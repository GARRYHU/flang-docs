8/** \file
9 * \brief Fortran unformatted I/O utility routines defined in unf.c and
10 * visible to the rest of the runtime.
11 *
12 * The calling sequence for these routines is always __f90io_unf_init(),
13 * __f90io_unf_read/write()... __f90io_unf_end(). __f90io_unf_init()
14 * initializes globals. __f90io_unf_read() reads one record into a
15 * buffer the first time it is called and then processes that record
16 * in subsequent calls. __f90io_unf_write() collects data in a buffer
17 * which is reallocated as needed in subsequent calls.
18 * __f90io_unf_end() either writes the buffer to the file in the case
19 * of a write, or seeks to the next record (if a variable length file)
20 * in the case of a read. If __fortio_error is called any time during or
21 * after an init, and is allowed to return (as opposed to exiting) by
22 * iostat, a global variable will be set and all subsequent calls to
23 * __f90io_unf_read() or __f90io_unf_write() and __f90io_unf_end()
24 * will return with an error. The next call to __f90io_unf_init()
25 * resets these flags.
26 */

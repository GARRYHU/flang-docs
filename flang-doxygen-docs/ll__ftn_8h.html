◆ print_entry_subroutine()

void print_entry_subroutine ( LL_Module *  module)

Write out all Entry's as a separate routine.

Each entry will call a master/common routine (MCR). The first argument to the MCR will determine which label(Entry) control will jump to upon entry into the MCR. If the MCR is a function, the next argument will be the function's return value. The next argument(s) will be all non-duplicate aggregate arguments for all entries. The MCR will always effectively be a subroutine.

int reference_for_temp ( int  sptr,

This routine finds the dimension of sptr.

It takes subscript a(f(i),5,f(j)). It eliminates scalar dimension. It makes an ast for reference sptr: a(f(i),5,f(j)) --> sptr(f(i),f(j))

NOTE: This is always called after get_temp_forall(), which calls mk_forall_sptr(). The subscripts are not always as simple as sptr(f(i),f(j)), especially if the stride is not known. if the stride is not +1 or -1, the subscript will be normalized.

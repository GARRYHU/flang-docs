◆ forall_2_sec()

static int forall_2_sec ( int  a,
int  forall 
)
static

This routine takes an array in a forall statement with its subinfo and replaces all forall indexes.

E.g., forall(i=1:10:2) a(i+1) will become a(2:11:2).

Note that this assumes each forall index appears in array subscripts. If not, something is wrong in the communication detection algorithm.

enum _DIST_TYPE

Data type codes.

These must correspond to the compiler's codes.

Any changes to the intrinsic types must get reflected in the F2003 type descriptors defined in type.c (see I8(__f03_ty_to_id)[] array). (see also __NTYPES below).

Detailed Description

Fortran symbol table access module.

FTN - symbol table access module. This module contains the routines used to initialize, update, access, and dump the symbol table. Note that in addition to being used by FTN, this module is used by the utility program, symini, which processes intrinsic and generic definitions in order to set up the initial symbol table for FTN.

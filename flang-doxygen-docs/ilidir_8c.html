◆ find_lpprg()

static int find_lpprg ( int  line)
static

Find the set of loop pragmas associated with a line number (the set's beginning and ending line numbers enclose the requested line number).

Since the loop sets occur in lexical order, the last loop set found is the set with beginning line number is the floor of the requested line number.

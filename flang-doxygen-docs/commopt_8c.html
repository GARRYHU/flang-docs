int  lp1,
int  nested 
)
static

... 

  S1: A(i) = ...
  S2: ...  = ... A(f(i)) ...

The loops cannot be fused if either of the following calls to dd_array_conflict() return TRUE: 

  dd_array_conflict(tripletList, A(f(i)), A(i), FALSE)
  dd_array_conflict(tripletList, A(i), A(f(i)), TRUE)

Alternatively, the statements within the original loops may be of the following form:

  S1: ...  = ... A(f(i))
  S2: A(i) = ...

The loops cannot be fused if the following call to dd_array_conflict() returns TRUE. 

  dd_array_conflict(tripletList, A(i), A(f(i)), FALSE)

The back end recognizes loops with 0 RHS and transforms these into calls to _mzero, so don't fuse assignments with just 0 on the RHS.

◆ visit_ilis()

bool visit_ilis ( ILI_visitor  visitor,
void *  visitor_context,
bool  context_insensitive 
)

ILI rewriting driver. Returns TRUE if any change occurs.

The traversal is in block sequence order, with the ILTs being visited in forward order within each block.

The void *visitor_context is passed to the visitation callback as at->context.

Set the "context_insensitive" flag to TRUE for faster processing when the rewriting visitor is known to not be sensitive to the context (i.e., it always maps ILI 'x' to the same 'y' in any statement).

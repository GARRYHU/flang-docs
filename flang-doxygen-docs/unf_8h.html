Detailed Description

Fortran unformatted I/O utility routines defined in unf.c and visible to the rest of the runtime.

The calling sequence for these routines is always __f90io_unf_init(), __f90io_unf_read/write()... __f90io_unf_end(). __f90io_unf_init() initializes globals. __f90io_unf_read() reads one record into a buffer the first time it is called and then processes that record in subsequent calls. __f90io_unf_write() collects data in a buffer which is reallocated as needed in subsequent calls. __f90io_unf_end() either writes the buffer to the file in the case of a write, or seeks to the next record (if a variable length file) in the case of a read. If __fortio_error is called any time during or after an init, and is allowed to return (as opposed to exiting) by iostat, a global variable will be set and all subsequent calls to __f90io_unf_read() or __f90io_unf_write() and __f90io_unf_end() will return with an error. The next call to __f90io_unf_init() resets these flags.

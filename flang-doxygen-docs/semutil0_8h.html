Issue error messages only for impossible conversions. Return constant value for 32-bit constants, or symbol table pointer. Can only be used for scalar constants.

Remember: Non-decimal constants are octal, hexadecimal, or hollerith constants which are represented by DT_WORD, DT_DWORD and DT_HOLL. Non-decimal constants 'assume' data types rather than go thru a conversion. Hollerith constants have a data type of DT_HOLL in the semantic stack, however, the symbol table entry they point to has a data type of DT_CHAR.

Hollerith constants are always treated as scalars while octal or hexadecimal constants can be promoted to vectors.

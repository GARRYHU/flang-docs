13/*
14 *
15 * Examples:
16 * + target is 64-bit
17 * + for 64-bit x86 targets, the C data type long is 64-bit except for WIN.
18 * + pointer-size integer type
19 * ...
20 * Typically, these features could be appear in the source code using
21 * conditional compilation based on certain combinations of TARGET_ checks.
22 * The idea is an appropriately named feature would be defined in this
23 * file and uses those same TARGET_checks.

30 * X86_32 => new 32-bit x86 CG
31 * I386 => old 32-bit x86 CG
32 */

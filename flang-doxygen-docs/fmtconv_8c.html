Detailed Description

Utility module for converting the internal representation of a data item to string form.

  • __fortio_default_convert() - conversion routine which applies the default editing rules with respect to a data type.
  • __fortio_fmt_i() - Iw, Iw.m edit descrs.
  • __fortio_fmt_g() - Gw.d, Gw.dEe
  • __fortio_fmt_e() - Ew.d, Ew.dEe, ENw.d, ENw.dEe, ESw.d, ESw.dEe,
  • __fortio_fmt_d() - Dw.d
  • __fortio_fmt_f() - Fw.d

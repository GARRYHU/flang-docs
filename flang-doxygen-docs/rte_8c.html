int get_header_member_with_parent ( int  parent,
int  sdsc,
int  info 
)

Generate an AST for accessing a particular field in a descriptor header.

Note: This is similar to get_header_member() above except it also operates on descriptors that are embedded in derived type objects.

Parameters
parentis the ast of the expression with the descriptor that we want to access. This is needed if the descriptor is embedded in a derived type object.
sdscis the symbol table pointer of the descriptor we want to access.
infois the field we want to access in the descriptor.
Returns
an ast expression of the descriptor access.

◆ sptr

int LL_Object::sptr

Symbol table reference. This value isn't used by ll_structure.c or ll_write.c. It could be used by the init function to locate the data for the object initializer.

Note that the lifetimes of Fortran symbol table entries are shorter than the lifetime of an LL_Module.

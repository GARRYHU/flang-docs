◆ enqueueTbp()

static int enqueueTbp ( int  sptr,

tbpTask  task 
)
static

Add type bound procedure (tbp) to tbp queue, tbpQueue.

If task is TBP_ADD_SIMPLE, then we are adding a tbp that was declared as "procedure tbp".

If task is TBP_ADD_IMPL, then we are adding a tbp that was declared as "bind => implementation".

If task is TBP_CHECK_CHILD, then we are also checking the validity of a child tbp.

If task is TBP_CHECK_PUBLIC, then we are also checking the validity of a public declared tbp.

If task is TBP_CHECK_PRIVATE, then we are also checking the validity of a private declared tbp.

Parameters
sptris the symbol table pointer of the implementation name
bindis the symbol table pointer of the binding name
offsetis tbp's offset in dtype's "virtual function table"
dtypeis the derived type record that we're currently processing
taskis the task that invoked this function
Returns
integer > 0 if successful, else 0

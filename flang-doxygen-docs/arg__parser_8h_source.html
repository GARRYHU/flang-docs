8/** \file
9 * \brief C header file for the argument parser
10 *
11 * Rules of the game:
12 * - Argument parser exepects usual argc, argv pair; tries to handle all
13 * arguments
14 * - Swithches start with a '-'
15 * - First or last argument without a leading dash is considered the input file
16 * name (only one), everything else is treated as a switch followed by zero
17 * or more values

63/** \brief Register a string list argument
64 *
65 * Add a "string list argument", that would produce a null terminated list of
66 * string pointers. Values are taken from argument pointers, but space for them
67 * needs to be allocated and freed by the consumer (argc can be a fair guess of
68 * the number of array elements to allocate).

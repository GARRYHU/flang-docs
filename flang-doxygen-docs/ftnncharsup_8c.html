Implements relational operators with string operands and the lexical intrinsics. Returns integer value:

  • 0 => strings are the same
  • -1 => a1 lexically less than a2
  • 1 => a1 lexically greater than a2 If the strings are of unequal lengths, treats shorter string as if it were padded with blanks.

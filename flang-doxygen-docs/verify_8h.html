◆ VERIFY_LEVEL

typedef enum VERIFY_LEVEL VERIFY_LEVEL

VERIFY_LEVEL specifies the depth to which to verify ILI.

Deeper levels require more time. A good strategy is to choose a depth for which the verification time can be amortized against compiler work that have happened since the last verification.

enum VERIFY_LEVEL

VERIFY_LEVEL specifies the depth to which to verify ILI.

Deeper levels require more time. A good strategy is to choose a depth for which the verification time can be amortized against compiler work that have happened since the last verification.

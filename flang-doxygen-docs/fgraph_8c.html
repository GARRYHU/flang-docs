◆ rdilts()

void rdilts ( int  fgx)

'read' a flowgraph node.

...

This defines the 0th STD entry to locate the beginning and end STDs of the node. The STD state (values of the STDs in the 0th entry and the beginning and last STDs of the flowgraph node) is saved. After a node has been read, its STD list has the following properties:

  1. the list is terminated by 0.
  2. STD_NEXT(0) locates the first STD in the list.
  3. STD_PREV(0) locates the last STD in the list.

◆ sub_types

const struct LL_Type_** LL_Type_::sub_types

The sub_types field depends on the value of data_type

  • LL_PTR [0] = pointee
  • LL_ARRAY [0] = element type, sub_elements = #elements
  • LL_VECTOR [0] = lane type, sub_elements = #lanes
  • LL_STRUCT [0..sub_elements-1] = members
  • LL_FUNCTION [0] = return type, [1..sub_elements-1] = argument types

◆ scale_of()

int scale_of ( DTYPE  dtype,
INT *  size 
)

...

...

This machine dependent routine computes the size of a data type in terms of two quantities:

  • size - number of elements in the data type (returned thru size).
  • scale - number of bytes in each element, expressed as a power of two (the return value of scale_of).

This routine will be used to take advantage of the machines that have the ability to add a scaled expression (multiplied by a power of two) to an address. This is particularly useful for incrementing a pointer variable and array subscripting.

Note that for those machines that do not have this feature, scale_of returns a scale of 0 and size_of for size.

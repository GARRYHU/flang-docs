static void assign1 ( int  rtype)
static

Assign registers for opt level 1.

The possible set of registers to use satisfies the relation first_global <= reg <= last_global, where the bounds are found in the mach_reg structure of the reg structure for the given register type. The registers are assigned beginning at last_global and continues down to first_global. The value returned by the function is the number of registers assigned.

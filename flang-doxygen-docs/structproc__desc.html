Detailed Description

Procedure Pointer/Argument Descriptor.

This descriptor holds the closure pointer of a procedure pointer or procedure dummy argument. This is used to support F2008 internal procedures passed as arguments and pointer targets.

NOTE: The fields in object_desc must be consistent in type and length with the fields in F90_Desc minus the F90_DescDim field (see also F90_Desc).

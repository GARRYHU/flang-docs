static void add_dfn_loop ( int  l,
int *  pn 
)
static

Reorder the loops in the LP_LOOP order.

Right now they are top-sorted according to the parent relationship, but otherwise unordered. Here we use a more constructive sort, so a loops children are contiguous to the loop. fill LP_DFN with DFN of loop, and LP_CHILDDFN with DFN of last child.
